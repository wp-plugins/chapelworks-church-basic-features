== Description ==
This plugin adds basic functions to church websites:
 - Church News Custom Post Type
 - Ministry Description Custom Post Type
 - Church Staff Custom Post Type
 - Ministry Category Custom Taxonomy
 - Church Information Administrative Panel 
 - Slideshow Support
 
The ChapelWorks Charis Church theme is available [here](http://wordpress.org/themes/charis-church/) and was designed to make optimal use of this plugin. 

== Installation ==
This Plugin is installed via the WordPress Plugin Installer. For detailed setup instructions, vist the ChapelWorks [Documentation](http://chapelworks-church-themes.com/documentation/) page.




== Frequently Asked Questions ==

= What other Church related plugins are available? =

There are additional plugins available that provide functions for church websites. The ChapelWorks Premium Church Plugin is designed to be used in combination with the Basic Church plugin and adds additional functions. It is included with premium themes from ChapelWorks.
